# Gastos-Inteligentes
Gastos Inteligente é uma aplicação web completa de gestão financeira pessoal desenvolvida com HTML5, CSS3 e JavaScript puro. O sistema permite que usuários controlem seus gastos mensais, visualizem dados através de gráficos interativos e recebam recomendações inteligentes de economia e investimento baseadas na regra 50-30-20.
